Title Project:
Classroom Reservation System Web-Application

This project aims to be a tool that allows users to conveniently reserve classrooms to make classroom management more efficient and faster. Users can manage and schedule classroom use.

Web development: HTML, CSS, Javascript, Django, GitHub and framework bootstrap.

Display the login page and enter the username and password to access the system.

The page after logging in allows you to select a room, date, and time for booking a classroom.

After selecting the desired room, date, and time, click save.
A notification will appear specifying the time to confirm whether you want to book the specified time.
Then, select the room you want to book by clicking the "Book" button to fill out the classroom booking form. If you want to search for a room, click the "Search" button.

After clicking the "Search" button, a window will appear for searching rooms.
You can type the room number in the search field to select the room you want to book.

After typing the room number in the search field, the selected room will be displayed. For example, if you type room number ท.539 in the search field, room ท.539 will be shown.

The form page will display the classroom location, number of seats, date, and time that were previously saved. There will also be fields to enter the name of the person booking, their email, and details about the purpose of booking the classroom.

When filling out the form The status page shows all booking details. Including the reservation's name, email address, room number, and date of arrival. You can click "Edit" to edit the information or click "Delete" to delete the booking.

When you click the "Edit" button, you'll be directed to a page where you can modify the booking details, including the name of the booker, their email, and the details used for the room reservation.

After clicking the "Edit" button and confirming the edited information, there will be a notification indicating that the data has been successfully updated.

When you click the "Delete" button, a confirmation message will appear asking if you want to delete the booking information.

When you click the "OK" button, the booking information will be deleted as shown in the image.

When you click the "Logout" button, a message will appear saying "User has logged out of the system." It will prompt you to enter your username and password to log in again, or you can click on the house icon to return to the home page.

POSTER:
Garden Facilities Management

This project involves the management of a backyard garden, encompassing efficient control and monitoring across various aspects of the garden or outdoor space. This includes tasks such as watering plants, adjusting lighting conditions, and maintaining environmental parameters like temperature and humidity. In your case, you are referring to a system that utilizes Line Bots, along with simulated hardware in Proteus, in conjunction with Arduino code and Python for command control.

Connection : Creating a LINE Bot that connects with Python, ngrok, and Arduino. In the Python part, the Flask and Line_bot_sdk libraries are utilized to establish a responsive system for the LINE Bot. Additionally, ngrok is employed to enable Python to receive messages from the LINE Bot.

Flowchart : The operational principles of the Garden Facilities Management system program.

Proteus : The creation of a simulation system for real-world usage in Garden Facilities Management.

Operation : There is an interactive communication with a LINE Bot for which code has been developed to execute specific commands as required, facilitating the management of activities within the garden.
