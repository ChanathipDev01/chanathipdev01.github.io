MINI PROJECT

POSTER:
Garden Facilities Management

This project involves the management of a backyard garden, encompassing efficient control and monitoring across various aspects of the garden or outdoor space. This includes tasks such as watering plants, adjusting lighting conditions, and maintaining environmental parameters like temperature and humidity. In your case, you are referring to a system that utilizes Line Bots, along with simulated hardware in Proteus, in conjunction with Arduino code and Python for command control.

Connection : Creating a LINE Bot that connects with Python, ngrok, and Arduino. In the Python part, the Flask and Line_bot_sdk libraries are utilized to establish a responsive system for the LINE Bot. Additionally, ngrok is employed to enable Python to receive messages from the LINE Bot.

Flowchart : The operational principles of the Garden Facilities Management system program.

Proteus : The creation of a simulation system for real-world usage in Garden Facilities Management.

Operation : There is an interactive communication with a LINE Bot for which code has been developed to execute specific commands as required, facilitating the management of activities within the garden.
